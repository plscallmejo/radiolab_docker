## radiolab_docker

Tools for constructing, managing, and deploying a robust and reproducible neuroimage analysis environment.

## Included softwares

* AFNI
* ANTs
* FSL
* Freesuefer
* convert3D
* dcm2niix
* miniconda (Radiolabconda environment with python3.7)

## Ongoing

Switching python docker API.
